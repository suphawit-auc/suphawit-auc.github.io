HTML Element

ในการเขียน HTML ต้องเริ่มต้นเเละจบด้วย Tag เปิดปิดเสมอ <tag>.....</tag>
โดย tag พื้นฐานที่ควรรู้เช่น <h> ใช้กำหนดว่าส่วนนี้เป็น head <p> ใช้บอกว่าส่วนนี้เป็น paragraph
การเริ่มเขียนเราต้องบอกประเภทข้อมูลก่อนโดยพิมพ์ว่า <!DOCTYPE html>
โค้ดทั้งหมดที่เราเขียนต้องอยู่ใน tag <html> เปิดปิด เเละส่วนของเนื้อหาที่จะเเสดงบน web ต้องอยู่ใน tag <body> เปิดปิด
ตัวอย่างโค้ดเช่น หากเราต้องการให้ web ขึ้นข้อความ Hello World เราก้ต้องเขียน Hello world อยู่ในส่วน body
<html> <body> <p> This is a paragraph </p> </body> </html>
