HTML Element

ในการเขียน HTML ต้องเริ่มต้นเเละจบด้วย Tag เปิดปิดเสมอ
โดยที่ element จะอยู่ใน tag
 <tag>.....</tag>

โดย element พื้นฐานที่ควรรู้เช่น
 <h> ใช้กำหนดว่าส่วนนี้เป็น head
 <p> ใช้บอกว่าส่วนนี้เป็น paragraph

การเริ่มเขียนเราต้องบอกประเภทข้อมูลก่อนโดยพิมพ์ว่า
 <!DOCTYPE html>
โค้ดทั้งหมดที่เราเขียนต้องอยู่ใน tag <html> เปิดปิด
เเละส่วนของเนื้อหาที่จะเเสดงบน web ต้องอยู่ใน tag <body> เปิดปิด

ตัวอย่างโค้ดเช่น หากเราต้องการให้ web ขึ้นข้อความ Hello World เราก้ต้องเขียน Hello world อยู่ในส่วน body 

            
            <html>
            <body>
            <p>
            Hello World
            </p>
            </body>
            </html>

HTML Attribute

Attribute คือตัวที่เอามารับค่าต่างๆซึ่งจะอยู่ข้างหน้า เเละ value ของมันจะอยู่ในเครื่องหมาย ""

Hyperlink

บนเว็บเราสามรถสร้าง hyperlink ได้โดยใช้ tag <a>

Visit My Website
            
            <a href="https://suphawit-auc.github.io/">Visit My Website</a>

ในช่อง href ให้ใส่ url ที่ต้องการเชื่อมเเละข้างหลังให้พิมพ์ข้อความที่ต้องการ

ใส่รูป

เราสามารถใส่รูปใน web ได้ด้วย tag <image> 

            
            <img src="/images/testimage.jpg" width="500" height="600">

ในช่อง src ให้ใส่ที่ที่เราเก็บรูปไว้ เเละสามารถกำหนดขนาดของรูปได้

Paragraph

Element <p> ไว้สำหรับเเบ่งข้อมูลเป็นย่อหน้าๆ

มี tag ที่เอาไว้ช่วยจัดหน้าเบื้องต้นดังนี้
 <hr>  ไว้ขีดเส้นขั้น
 <br>  ไว้ขึ้นบรรทัดใหม่
 <pre>  ไว้พิมพ์ข้อความโดย web จะเเสดงผลตามที่เราจัดหน้าไว้ใน pre เลย

Style

หากเราเพิ่ม Style ไปใน Element ต่างๆเราจะสามารถปรับเเต่ง ขนาดตัวอักษร Font สี เเละอื่นๆ ได้
ยกตัวอย่างหากเราต้องการให้ Paragraph ถัดไปตัวอักษรสีชมพูเเละพื้นหลังสีดำ

            
            <p style="background-color:black;color: pink; font-size: 24;">ทดลองจ้า</p2>

Formatting

มี element ที่เราสามารถนำมาเปลี่ยนรูปเเบบตัวอักษรได้เช่น

 <small>  ทำตัวเล็ก
 <strong>  ทำตัวเข้ม
 <del>  ไว้ขีดฆ่า

Comment

ในการเขียน code เราสามารถเพิ่ม comment ไปได้โดยอะไรก็ตามที่เขียนใน comment จะไม่เเสดงใน browser 

   <!-- คอมเม้นตรงนี้ -->

CSS

CSS ย่อมาจาก Cascading Style Sheets ใช้เพื่อช่วยจัดการเเละตกเเต่งหน้าเว็บ
โดยที่เราสามารถเรียกใช้ได้ 3 เเบบ

 1 เเบบ Inline กำหนด style เฉพาะของ element นั้นๆ โดยกำหนดไว้ใน tag เปิด

 <h1 style="color:blue;">A Blue Heading</h1>

 2 เเบบ Internal กำหนด style ไว้ใน head โดยการกำหนดเเบบนี้จะทำให้ element ที่เรากำหนดไว้เปลี่ยนไปทังหมด

 <style>
  body {background-color: powderblue;}
 </style>

 3 เเบบ External คือการสร้างไฟล์ sheet ของ CSS ที่กำหนดสิ่งต่างๆไว้ข้างนอกเเละ link มาใช้ใน head

Class

Class คือสิ่งที่ระบุ Object ที่มี element เหมือนๆกัน
ในหนึ่ง class จะมีได้หลาย element

ยกตัวอย่างเช่นเราต้องการเปลี่ยนสีตัวอักษรเเละพื้นหลังพร้อมกันเราก็สร้าง class ขึ้นมา 

            สร้าง style ขื่อ .test
            
            .test{
                background-color:orange;
                color: black;
              }

เมื่อเราต้องการเรียกใช้ class นี้เราก็เเค่ใช่ element
 <div class="test">


 </div>
