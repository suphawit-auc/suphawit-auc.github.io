Introduction

ในการเขียน Java Script เราต้องเขียนใน Tag Script เสมอ

 <script>document.getElementById("demo").innerHTML = "Hello World"</script>

เราสามารถเรียกใช้ได้ทั้งในส่วน Head เเละ Body เช่น

            <script>
                function testFunction() {
                  document.getElementById("demo").innerHTML = "เปลี่ยนเเล้ว";
                }
            </script>
            <p id="demo">ยังไม่เปลี่ยน</p>
            <button type="button" onclick="testFunction()">ลองกดดู</button>

ยังไม่เปลี่ยน

Output

เราสามารถเเสดงผล Java script ได้ 4 วิธี

inner HTML

โดย ID จะเป็นตัวกำหนด HTML ELement เเละกำหนดเนื้อหาใน innerHTML

    <p id="plustest"></p>
    <script>
    document.getElementById("plustest").innerHTML = 5 + 6;
    </script>

โดยโปรเเกรมก็จะเเสดงผลบวกออกมาตรงที่เรา p id = plustest

document.write()

เเสดงผลผ่าน document.write() โดยวิธีนี้มีข้อเสียคือหากเรียกใช้งานหลังจาก Document โหลดจะไปเขียนทับข้อมูลอื่น

    <script>
    document.write("สวัสดีครับ");
    </script>

window.alert()

วิธีนี้จะเเสดงผลเป็น Window popup ขึ้นมา

    <script>
    window.alert("สวัสดีครับ");
    </script>

console.log()

เเสดงผลผ่าน console log

 
    <script>
    console.log("สวัสดีครับ");
    </script>

Comment

ใน Java Script เราสามารถเขียน comment ได้หลัง //

หรือจะ comment หลายบรรทัดโดยเขียนใน /*...*/ ก็ได้

Variable

ใน Java Script เราสามารถประกาศตัวเเปรได้หลัง Var ตัวอย่างเช่น

 var x = 5;
 var person = "Prame";
